Phytoprom is a mining, identification and enrichment analysis tool for candidate cis-regulatory elements (CCREs) in plant genomes available on Phytozome database. It is a user-friendly web application, providing a way to characterize the large-scale gene datasets such as those from transcriptome analysis. The present platform is web-based and easy to handle graphical-user interface, which allows users to readily input gene lists with Phytozome IDs. Additionally, Phytoprom makes it possible to download sets of species-specific promoters, transcripts and proteins under study for analysis in other databases.

For more information, see documentation section.
