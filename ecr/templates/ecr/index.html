Phytoprom is an enrichment analysis tool for candidate cis-regulatory elements (CCREs). It is based on plant gene promoters available on Phytozome database. Phytoprom is a user-friendly web application, providing a way to characterize large-scale gene promoter datasets. The tool also makes it possible to download sets of species-specific promoters, transcripts and proteins under study for analysis in other databases.

For more information, see documentation section.

Graphical output
  • Visual output for rapid “at a glance” insight of enriched CCREs distribution in the scrutinized sample;
  • Online chart creation in .pdf format for data publication on scientific journals.
Tabular output
  • The tabular output contains detailed information of the CCREs (enriched or not) analyzed;
  • The table shows the found motifs, associated transcription factor (TF) and statistical data;
  • Also presents links to the scientific manuscripts with information about the role of the associated TF on plants under stress conditions.
